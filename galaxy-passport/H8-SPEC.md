## 0. 한눈에 (가장 중요한 것)

| | 접었을 때 (Cover/커버 화면) | 펼쳤을 때 (Main/메인 화면) |
|---|---|---|
| **크기** | 5.5″ | 7.6″ |
| **화면 비율** | **16:10 → 세로로 세우면 10:16 (세로로 긴 모양)** | **4:3 (거의 정사각에 가까운 가로 화면)** |
| **성격** | "숏폼(세로 영상)을 꽉 찬 화면으로" | "롱폼(영화·드라마)을 몰입감 있게 가로로" |
| **우리 경험에서** | 여권 만들기 = **세로 단계** (인사·미션·셀피·목적지 고르기) | 워프·시네마틱·와이드 셀피·최종 여권 = **가로 단계** |

> ⚠️ **정정:** 예전에 제가 "접으면 4:3, 펴면 16:9"로 알고 있었는데 **반대**였습니다.
> 실제로는 **접으면 10:16(세로로 긴 폰 모양), 펴면 4:3(넓은 화면)** 입니다.
> 이 값으로 앱 전체를 맞췄습니다. (`foldThreshold`, 스테이지 비율 모두 수정 완료)

## 1. 기기 라인업 (덱은 3종을 함께 다룸)

- **Galaxy H8** — *우리 프로젝트의 주인공.* 가장 가벼운 Z Fold. "콘텐츠에 최적화된 화면비"가 셀링 포인트.
- **Galaxy Q8** — 더 큰 화면(메인 8.0″ / 커버 6.5″)의 상위 폴드. 멀티태스킹·카메라 강조.
- **Galaxy B8** — 가장 얇은 플립(Flip). 커버 화면(FlexWindow) 활용, 휴대성 강조.

*(Q8·B8는 참고용. 아래 상세는 H8 중심.)*

## 2. Galaxy H8 상세 스펙 (덱 7페이지)

- **디스플레이**
  - 메인(펼침): **7.6″ QHD+ · 4:3**
  - 커버(접힘): **5.5″ QHD+ · 16:10**
  - Dynamic AMOLED 2X · **120Hz** 가변 주사율
  - **최대 3,000 nits**(Vision Booster) — 야외 시인성 우수, **완전히 평평하게 펴지는(lie-flat)** 화면
- **카메라**
  - 후면: **50MP 초광각(F1.9) + 50MP 광각(F1.8)** 듀얼
  - 전면: **10MP F2.2 (커버)** + **10MP F2.2 (내부/메인)** — 접든 펴든 셀피 가능
- **성능:** Snapdragon 8 Elite Gen 5 for Galaxy (3nm)
- **메모리/저장:** 12GB + 256/512GB, 1TB
- **배터리/충전:** 4,800mAh · 초고속충전 2.0 · 고속 무선충전 2.0
- **크기/무게:** 펼침 123.9 × 161.4 × 4.5 mm / 접힘 123.9 × 81.9 × 9.7 mm / **201 g**

### H8 대표 UX 기능 (덱 8~10페이지)
1. **두 화면, 콘텐츠별 최적화** — 커버(세로)로는 숏폼을 꽉 차게, 메인(가로 4:3)으로는 영화·드라마를 몰입감 있게.
2. **밝고 평평한 메인 화면** — 3,000 nits + lie-flat로 야외에서도 선명, 펼치면 눕혀서 보기 좋음.
3. **Auto-frame(자동 프레이밍)** — 영상 편집 시 피사체 움직임을 자동으로 잡아 크롭. SNS용으로 바로 편집.

## 3. 이게 우리 "Galaxy Passport" 흐름과 어떻게 맞는가

H8의 핵심 메시지 = **"접으면 세로 숏폼, 펴면 가로 몰입"**.
우리 경험은 정확히 이 **접다 → 펴다** 물리 동작에 스토리를 얹습니다.

```
┌──────────────────────────────────────────────────────────────┐
│  [1] 접힌 상태 · 커버 화면 (10:16 세로)  ─  "여권을 만드는 단계" │
│   · 인사 / H8·무엇이든 질문                                     │
│   · 미션 시작 → 여권 사진(셀피1) 촬영                            │
│   · 오브(구体) 룰렛에서 목적지 선택                              │
│   · "이제 H8을 펼쳐줘!" 안내                                    │
│                                                                │
│         ↓↓↓  실제로 기기를 펼침 (viewport 4:3로 전환 감지)  ↓↓↓ │
│                                                                │
│  [2] 펼친 상태 · 메인 화면 (4:3 가로)  ─  "여행을 떠나는 단계"   │
│   · 워프 카운트다운 + 시네마틱 (가로 몰입)                       │
│   · 목적지 테마 와이드 셀피(셀피2) 촬영                          │
│   · 나만의 4K "Galaxy Passport" 완성 (종이 질감 기념품)          │
│                                                                │
│         ↓↓↓  잠시 후: "H8을 접어주세요" (다음 여행자용)  ↓↓↓    │
│                                                                │
│  [3] 다시 접힘 → 처음(인사)로 리셋                              │
└──────────────────────────────────────────────────────────────┘
```

**왜 이 매핑이 좋은가:** 세로 커버에서 "준비(여권 발급)"를, 가로 메인에서 "몰입(여행/시네마틱)"을 하는 건
H8이 매장에서 실제로 파는 스토리("두 화면, 각기 다른 콘텐츠")를 **손으로 직접 체험**하게 만드는 것.
펼치는 순간이 곧 "이륙"이 됩니다.

## 4. 전체 스텝 (현재 구현 기준)

**세로(접힘, 10:16)**
1. 부팅 스플래시 → 디지털 휴먼 연결
2. 아이들: DG 대기 + 하단 "여행을 떠나볼까?" (제목 로고는 머리와 안 겹치게 하단 배치)
3. 인사: "반가워, 여행자!" + 두 선택지 — [💬 H8·무엇이든] / [🧭 어드벤처 미션]
4. (미션) 여권 발급 안내 → 여권 사진(셀피1) 촬영
5. 목적지 룰렛(오브 5개: 파리·제주·조선·미래도시·판타지월드) — 두 번째 탭 = 버블처럼 팡 터지며 선택
6. "이제 H8을 펼쳐줘!" (게이트 오픈 안내)

**펼침 감지 → 가로(4:3)**
7. 워프 라인 가속 + 카운트다운 → 시네마틱 (목적지 테마)
8. 변신 와이드 셀피(셀피2) 촬영
9. 3D 여권 책이 등장 → 표지가 (오른쪽 경첩으로) 열리며 사라짐 → 은하 속지 공개
10. 최종 종이 여권(4K 기념품): 여권사진·와이드셀피·스탬프·스티커·이름 편집·QR
11. "미션 끝" (스탬프 버튼) → 대화 계속 초대

**마무리**
12. 일정 시간 대기 → "H8을 접어주세요" (다음 여행자) → 접으면 처음으로 리셋

## 6. 변경 이력 (이번 정리로 반영됨)

- 화면비 정정: **접힘 10:16 / 펼침 4:3** (이전엔 4:3 / 16:9로 잘못 잡혀 있었음).
- `foldThreshold` = 1.0 (가로 세로 비율 1을 기준으로 접힘/펼침 자동 판별).
- 최종 종이 여권을 4:3에 맞게 재배치 — **글자는 사진과 절대 안 겹치게**, 스티커만 살짝 겹침 허용.
- 여권 표지: **오른쪽 경첩**으로 뷰어 쪽을 향해 열린 뒤 **바로 사라짐(페이드)**.
- 선택 버블/버튼: **불투명**으로 변경.
- 아이들 제목 로고: 머리와 겹치지 않도록 **하단 배치**.
